PERGUNTAS FREQUENTES

Como cadastrar produção?
Para registrar um novo ciclo no sistema, siga estas etapas simples:
1. Acesse o Menu: Clique na aba Ciclos de Produção localizada no painel principal.
2. Inicie o Cadastro: Selecione o botão + Adicionar Ciclo.
3. Preencha os Dados: Insira as informações solicitadas no formulário (como datas, insumos ou identificação).
4. Finalize: Clique em Salvar para concluir o registro.
Como gerar relatórios?
Após cadastrar sua produção na página de Ciclos de Produção, siga estas etapas para gerenciar seus documentos:
1. Acessar os Detalhes da Produção
Na listagem de ciclos, localize a produção desejada e clique no botão Ver Detalhes. Uma nova aba será aberta com as informações completas.
2. Escolher a Ação Necessária
Na tela de detalhes, você encontrará três opções principais:
Gerar Relatório: Utilize esta opção se for a primeira vez que você está processando os dados deste ciclo.
Ver Relatório: Escolha esta opção caso o relatório já tenha sido gerado anteriormente.
Deletar: Utilize para remover registros, se necessário.
3. Localizar Relatórios Salvos
Todos os relatórios gerados ficam armazenados de forma centralizada para facilitar a consulta futura:
1.Retorne à Página Principal do sistema.
2.Clique no botão Relatórios.
3.Nesta página, você encontrará o histórico de todos os documentos que você salvou.
O que é ciclo de produção?
Um ciclo de produção é o conjunto de etapas que uma empresa ou atividade realiza para transformar matéria-prima em produto final. Esse processo começa com a obtenção dos recursos (como sementes, materiais ou insumos), passa pela produção em si (cultivo, fabricação ou criação) e termina quando o produto está pronto para ser vendido. Resumindo: é o “caminho” que um produto percorre desde o início até chegar ao consumidor.
Como entrar em contato com o suporte?
Para falar conosco, role a página até encontrar o botão azul Falar com o Suporte. Ao clicar, basta selecionar a opção de contato que preferir.
