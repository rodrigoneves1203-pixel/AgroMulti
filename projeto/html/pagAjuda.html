Perguntas Frequentes

Como cadastrar uma produção?
Para registrar um novo ciclo no sistema, siga estas etapas simples:

1. Acesse o menu: clique na aba Ciclos de Produção, localizada no painel principal.

2. Inicie o cadastro: selecione o botão + Adicionar Ciclo.

3. Preencha os dados: informe as informações solicitadas no formulário, como datas, insumos e identificação da produção.

4. Finalize: clique em Salvar para concluir o registro.
Como gerar relatórios?
Após cadastrar sua produção na página de Ciclos de Produção, siga estas etapas para gerenciar seus documentos:

1. Acesse os detalhes da produção
Na lista de ciclos, localize a produção desejada e clique no botão Ver Detalhes. Uma nova página será aberta com todas as informações do ciclo.

2. Escolha a ação desejada
Na tela de detalhes, você encontrará três opções principais:

Gerar Relatório: utilize esta opção caso seja a primeira vez processando os dados deste ciclo.

Ver Relatório: escolha esta opção caso o relatório já tenha sido gerado anteriormente.

Deletar: utilize esta opção para remover registros, se necessário.

3. Localize relatórios salvos
Todos os relatórios gerados ficam armazenados de forma centralizada para facilitar consultas futuras:

1. Retorne à página principal do sistema.

2. Clique no botão Relatórios.

3. Nesta página, você encontrará o histórico de todos os documentos salvos.
O que é um ciclo de produção?
Um ciclo de produção é o conjunto de etapas realizadas para transformar matéria-prima em um produto final. Esse processo começa com a obtenção dos recursos necessários, como sementes, materiais ou insumos, passa pela produção em si (cultivo, fabricação ou criação) e termina quando o produto está pronto para ser vendido.

Resumindo: é o caminho que um produto percorre desde o início da produção até chegar ao consumidor.
Como entrar em contato com o suporte?
Para falar conosco, role a página até encontrar o botão azul Falar com o Suporte. Ao clicar nele, basta selecionar a opção de contato desejada.
