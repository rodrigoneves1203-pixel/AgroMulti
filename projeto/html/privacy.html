Política de Privacidade

O AgroMulti respeita a sua privacidade e se compromete a proteger os dados pessoais dos usuários.

1. Coleta de dados

Coletamos apenas dados necessários para o funcionamento do sistema, como email e informações de acesso.

2. Uso das informações

Os dados são utilizados exclusivamente para autenticação e operação da plataforma AgroMulti.

3. Segurança

Adotamos medidas de segurança para proteger os dados contra acesso não autorizado.

4. Compartilhamento

Não compartilhamos dados pessoais com terceiros.

5. Contato

Em caso de dúvidas, entre em contato com o suporte do AgroMulti.
